# Student Marks Analyzer

A simple Python project that analyzes student marks using the NumPy library.  
This program accepts marks from multiple students, stores them in a NumPy array, and calculates useful statistics like total marks, average, highest marks, lowest marks, and grades.

---

## Features

- Accepts marks for multiple students
- Stores marks using NumPy arrays
- Calculates:
  - Total marks
  - Average marks
  - Highest marks
  - Lowest marks
- Assigns grades based on average marks
- Uses conditional statements for grading logic

---

## Technologies Used

- Python
- NumPy

---

## Grading Logic

| Average Marks | Grade |
|---------------|-------|
| Above 85      | A     |
| 70 – 85       | B     |
| 50 – 69       | C     |
| Below 50      | Fail  |
